# Chrome Extension For LeetPath
LeetPath Chrome extension brings the full functionality of the LeetPath website directly to your browser. It offers personalized LeetCode problem recommendations based on your solved questions

## üöÄ Deployment and Hosting

- **Local Deployment**:
  - Clone the repository: 
    ```bash
    git clone https://https://github.com/amri-tah/LeetPath
    ```
  - Navigate to the project directory:
    ```bash
    cd LeetPath/chrome_extension
    ```
  - Install dependencies:
    ```bash
    npm install
    ```
  - Build the project:
    ```bash
    npm run build
    ```
  - Load the extension in Chrome:
    1. Open `chrome://extensions/` in your browser.
    2. Enable **Developer mode**.
    3. Click on **Load unpacked** and select the `dist` folder in the project.
  - The extension will now be available for use in your browser locally!
